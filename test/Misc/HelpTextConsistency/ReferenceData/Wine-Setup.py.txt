usage: Wine-Setup.py [-h] [-p PATH] [-v] [-V]

Initial configuration of Wine, e.g. for cross-compilation with BST.py.

optional arguments:
  -h, --help            show this help message and exit
  -p PATH, --path PATH  config directory (default: $HOME/.wine)
  -v, --verbose         show debug messages
  -V, --version         show version info and exit

examples:
  Wine-Setup.py

Please report bugs on GitLab (https://dmz-gitlab.honda-ri.de/TECH_Team/ToolBOSCore/-/issues).
