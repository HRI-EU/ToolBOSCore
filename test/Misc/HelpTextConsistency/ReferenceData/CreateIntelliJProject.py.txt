usage: CreateIntelliJProject.py [-h] [-v]

Turns a software package into a project for the JetBrains IntelliJ
IDE.

  -h, --help     show this help message and exit
  -v, --verbose  show debug messages

examples:
  cd MyPackage/1.0
  CreateIntelliJProject.py
