usage: FindProxyInstallations.py [-h] [-v] [-V]

Lists all packages installed only in your SIT sandbox ("proxy
directory").

optional arguments:
  -h, --help     show this help message and exit
  -v, --verbose  show debug messages
  -V, --version  show version info and exit

examples:
  %prog
